Volcano is a JavaScript query API for Obsidian. You can read the contents of your vault as structured data. The primary use-case is to use Obsidian as a content management system in a Node.js, Deno, or Bun-backed website.
## Features

- Open your notes as structured data
- Resolves wiki-links
- Verifies frontmatter according to schemas
- Resizes images
- Shortcodes for inserting content programmatically
